Автоматизированная систама для отслеживания задач и проектов для Naumen консалтинг

Это приложение поможет Вам с отслеживанием задач.

Для большинства функций требуется вход в систему. Неавторизованные пользователи имеют доступ к экрану приветствия и панели входа или регистрации.

Администратор (менеджер) может:

  • Создать задачу и назначить задачу любому пользователю
  • Посмотреть список пользователей и возможностью удалить пользователя
  • Просмотреть список всех задач с редактированием или удалением задачи

Обычный пользователь (сотрудник) может:

  • Создать задачу только для себя
  • Просмотреть список всех пользователей без доступа к редактированию
  • Просмотреть список всех задач, но отредактировать или удалить только задачи, за которые он отвечает

Каждый авторизованный пользователь может:

  • Посмотреть собственный профиль
  • Сменить статус задачи на выполненна/незаконченна
